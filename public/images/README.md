# Project images

Place the following image files in this folder so the Projects section and favicon load correctly:

- **MiniGames.jpeg** – JavaMiniGames project
- **ATM System.jpg** – ATM Simulation System project  
- **Network Analyzer.jpg** – Network Info Using Java project
- **SR-logo.jpg** – Favicon (optional; used in `index.html`)

Use these exact filenames (including spaces where shown) so they match the paths in `src/pages/Projects.tsx`.

In Vite, files in `public/` are served from the site root, so these will be available at `/images/<filename>` in development and in production builds.
